# 기획 의도
유니티 게임 개발에 들어가면 빈번하게 다룰 Update를 통한 실시간 게임 구현을 미리 연습해보기 위해서
# 핵심 플레이 흐름
좌측 중앙에서 시작. 우측에서 적이 나오고 이를 쏘아 처치하면 점수를 얻음. 모든 적이 처치하거나 지나가면 다음 스테이지 시작. 최종 스테이지의 보스를 처치하면 클리어. 적의 탄환에 적중하거나 적 기체와 부딪히면 게임오버
