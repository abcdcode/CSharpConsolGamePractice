# 개요
사이드뷰 슈팅 게임
C# 콘솔 게임 프로젝트. 비동기 메소드를 활용한 Update로 실시간 화면 갱신 및 2D 좌표계 구현을 통해 유니티 2D 환경을 모방한 학습용 게임

ConsoleShootingGame\bin\Debug\net10.0\ConsoleShootingGame.exe 를 실행해서 플레이

권장 화면 사이즈 있음. 안맞추면 UI 망가짐
# 기획 의도
유니티 게임 개발에 들어가면 빈번하게 다룰 Update를 통한 실시간 게임 구현을 미리 연습해보기 위해서



는 핑계고 사실 그냥 슈팅 게임이 갑자기 만들고 싶어져서
# 핵심 플레이 흐름
좌측 중앙에서 시작. 우측에서 적이 나오고 이를 쏘아 처치하면 점수를 얻음. 모든 적이 처치하거나 지나가면 다음 스테이지 시작. 최종 스테이지의 보스를 처치하면 클리어. 적의 탄환에 적중하거나 적 기체와 부딪히면 게임오버
# 주요 기능
## 플레이어 - 조작법
상하좌우 방향키로 이동.
Z키로 공격, X키로 폭탄 사용
## 스테이지 구성
### 적
정해진 시점에 정해진 위치에서 적이 생성되어 좌측으로 이동하며 플레이어를 공격함. 적을 처치하면 점수를 획득
적의 행동 패턴은 적과 스테이지에 따라 제각각
### 아이템
일부 적은 처치 시 아이템을 드랍함. 아이템은 대각선으로 움직이며 아이템 근처로 플레이어가 이동하면 아이템을 획득하여 능력치가 상승
일정시간내에 미획득 시 아이템은 사라짐
- P : 공격력 증가
- S : 이동속도 증가 
- R : 공격속도 증가
